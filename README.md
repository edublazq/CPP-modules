# CPP Modules (42 School)

Este repositorio está pensado para los **módulos de C++ de 42**.

En 42, los CPP Modules son una progresión de ejercicios para aprender C++ (normalmente en estándar C++98), pasando de lo básico a conceptos más avanzados como herencia, polimorfismo, plantillas y contenedores.

## ¿De qué va cada módulo?

- **Module 00**: Introducción a C++, clases básicas, funciones miembro, namespaces, iostream y compilación.
- **Module 01**: Gestión de memoria, referencias, punteros a miembros y uso inicial de `new`/`delete`.
- **Module 02**: Forma canónica ortodoxa (constructores/copia/asignación/destructor) y sobrecarga de operadores.
- **Module 03**: Herencia.
- **Module 04**: Polimorfismo por subtipado, clases abstractas e interfaces.
- **Module 05**: Excepciones y manejo robusto de errores.
- **Module 06**: Conversión de tipos (`static_cast`, `dynamic_cast`, etc.) y serialización básica.
- **Module 07**: Plantillas (templates) genéricas.
- **Module 08**: Templates aplicados a contenedores STL, iteradores y algoritmos.
- **Module 09**: Contenedores de la STL y resolución de problemas con estructuras de datos.

## Objetivo global

Dominar los fundamentos de C++ escribiendo código orientado a objetos, seguro y mantenible, siguiendo las reglas de evaluación de 42.
